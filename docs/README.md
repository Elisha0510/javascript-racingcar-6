# 2주차 프리코스 미션 - 자동차 경주 🚗

## 기능 목록
1.  [O] 사용자에게 자동차 이름을 입력 받는다.

* 1번 - 예외 처리
    * [O] 자동차 이름이 5자 초과인 경우

2. [O] 사용자에게 시도할 횟수를 입력 받는다.

* 2번 - 예외 처리
    * [O] 숫자가 아닐 경우
    * [O] 공백이 포함될 경우
        
3. [O] 각 자동차마다 랜덤값를 구한다.
4. [O] 구한 랜덤값에 따라 전진/멈춤한다.
5. [O] 사용자에게 실행결과를 보여준다.

* 3, 4, 5번을 시도할 횟수에 따라 반복한다.

6. [O] 최종 우승자를 구한다.
